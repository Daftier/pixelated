# Pixel Runner

A simple browser-based pixel game starter built with HTML, CSS, and JavaScript.

## How to run

1. Open `index.html` in your browser.
2. Use the arrow keys to move the hero.
3. Collect coins and avoid the red obstacle.

## Files

- `index.html` — game layout and canvas element
- `style.css` — pixel-inspired styling
- `script.js` — game logic and movement
